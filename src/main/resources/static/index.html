Välkommen till Julias kodprov för FinFollow!

Följande endpoints finns tillgängliga:

  • login/bankid - Genererar en QR-kod. Scannas den kan man logga in på Avanza via Mobilt BankId.
  • login/bankid/status - Visar status för inloggningen. Vid completed visas istället inloggnings-sessionen.
  • account/overview - Om inloggad visas information om alla ens konton på Avanza. Annars visas ett felmeddelande.
